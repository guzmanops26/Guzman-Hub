Privacy Policy

Guzman Hub — effective July 2026 (v1.0)

What this application is

Guzman Hub is a private, personal-use financial dashboard operated by its sole owner, Javier Guzman. It has exactly one user: the owner. It is not offered to the public, has no customers, and serves no third parties.

What data is collected

With the owner's authorization through Plaid, the application retrieves financial data for accounts the owner holds: account balances, transaction history, and liability details (such as credit card balances and due dates). The application collects no data about anyone other than the owner.

How data is used

Data is used solely to display the owner's finances to the owner: spending categorization, cash-flow tracking, subscription detection, and debt monitoring. It is never used for advertising, profiling, or any commercial purpose.

Sharing and selling

No data is sold, rented, or shared with any third party. Data moves between exactly two parties: Plaid (the authorized data network) and this application's private database.

How data is protected

  • All data in transit is encrypted with TLS 1.2 or higher.
  • All data at rest is encrypted (AES-256) in a private database with row-level security enabled and deny-by-default access.
  • Access to this application requires passkey authentication with biometric or device-PIN verification. No passwords exist on this system.
  • The application holds read-only financial data access and cannot move funds.

Data retention and deletion

Data is retained while the Hub is in service, for the owner's historical analysis. The owner may delete any or all data at any time, and may revoke Plaid's access to any linked institution at any time via Plaid's tools or the institution directly.

Plaid

Bank connections are made through Plaid. Plaid's handling of data is described in its own end-user privacy policy at plaid.com/legal/#end-user-privacy-policy.

Contact

Questions about this policy: the owner's operations email, on file with Plaid as this application's support contact.
